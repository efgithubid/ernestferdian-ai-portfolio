# Financial News Sentiment Analysis (Indonesia Focus)

## Overview
This project applies Natural Language Processing (NLP) to classify Indonesian financial news into sentiment categories such as positive, neutral, and negative. The goal is to explore how sentiment signals from financial news can support business insights, market monitoring, and future AI-powered financial applications.

## Business Problem
Financial news influences investor behavior and market sentiment. Manually reading and interpreting large volumes of financial news is slow and difficult to scale. This project explores whether machine learning and NLP can be used to automatically classify sentiment in Indonesian stock-related news.

## Objective
The objective of this project is to build and evaluate an NLP model that can predict the sentiment of Indonesian financial news articles. In addition to model performance, the project also considers how the output could be used in practical settings such as market intelligence, investment research, or sentiment dashboards.

## Dataset
The dataset contains Indonesian financial news text labeled by sentiment:
- Positive
- Neutral
- Negative

This project focuses on text preprocessing, model training, evaluation, and interpretation of business relevance.

## Methodology
The project follows these main steps:

1. Data loading and inspection  
2. Text cleaning and preprocessing  
3. Feature extraction / text representation  
4. Model training  
5. Evaluation using classification metrics  
6. Interpretation of results and business implications
